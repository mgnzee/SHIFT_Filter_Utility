Курс Java. Тестовое задание.

Утилита фильтрации содержимого файлов.

Java 25.0.2

Инструкция по запуску:  
Работа программы приближена к стандартным консольным утилитам Linux. Ввод/вывод из stdio, поддерживаются конвейеры.
Сборка:  
javac Filter.java  
jar cfe filter.jar Filter Filter.class  
Как пользоваться:  
java -jar filter.jar --help  
  
Особенности реализации, не уточненные в задании.  
Флаги:  
--help краткая инструкция по пользованию программой;  
-A перезаписывать файлы;  
-b и -B пропускать и оставлять пустые строки соответственно;  
-q не отображать статистику.  
Поддерживаются множественные флаги (например -afb), если флаги противоречат друг другу, применяется последний по очереди.  
Системы сборки, сторонние библиотеки не использовались.
